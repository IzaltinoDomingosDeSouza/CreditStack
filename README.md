# CreditStack

> **Own your credit.** A privacy-first credit management dashboard with zero-knowledge architecture.

CreditStack provides a unified view of your cards and transactions without the privacy tax. No bank logins, no background syncingâ€”just total control over your limits, spending, and due dates.

---

## Features

* **Unified View:** Aggregate all your credit cards in one place.
* **Zero-Knowledge:** Your data is encrypted locally. We have no access to your financial life.
* **No Third-Party Sync:** Skip the bank credentials.
* **Smart Limits:** Visual tracking of credit utilization across all accounts.
* **Due Date Alerts:** Localized notifications to ensure you never pay a late fee.

## Development Setup

### Prerequisites

* **Node.js**
* **React Native CLI**

### Quick Start

1. **Clone the repo**
```bash
git clone https://github.com/IzaltinoDomingosDeSouza/CreditStack.git
cd CreditStack

```


2. **Install Dependencies**
```bash
npm install
# or
yarn install
```

3. **Launch the Application**
```bash
npx expo start
```

## Privacy & Security

CreditStack is designed to be "Trustless."

* **Local-Only:** Data is stored using device-level encryption.
* **Anonymity:** No account creation is required to use the core features of the app.
* **Data Portability:** You own your data. Export your transaction history as JSON or CSV anytime.
